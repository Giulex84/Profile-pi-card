# Identity Card

Identity Card is a Pi Network dApp that provides users with a digital
identity and reputation preview.

## Features
- Pi Network authentication
- Identity card generation
- Reputation progression
- Daily engagement system
- Future-ready ecosystem vision

## Data & Privacy
No personal data is collected or stored.
All authentication is handled by Pi Network.

## Roadmap
Future versions may include verified badges,
ecosystem integrations and optional premium features.
