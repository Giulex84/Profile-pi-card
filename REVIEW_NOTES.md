What the App Does (v1.0.0)

User authentication via Pi Browser

Profile creation using non-sensitive fields only

Public, shareable profile page

Ability to publish or unpublish the profile at any time

User data export and full account deletion

Abuse reporting functionality

What the App Does NOT Do

No payments of any kind

No monetization features

No “coming soon” features (now or in the future)

No user rankings, messaging, or social feeds

No collection of email addresses or phone numbers
